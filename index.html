About

Scott de Jonge is a UI/UX designer & Web developer that designs and builds websites for humans at bigfish.tv.

Scott de Jonge has a Bachelor of Multimedia Design from the University of Queensland with a focus on creating universally accessible reponsive web interfaces for anyone, anywhere.

Turret

Sible is a web based tool for discovering location accessibility information in a geospacial context.

An icon font for use with Google Maps API and Google Places API using SVG markers and icon labels.
